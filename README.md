An OpenAI-powered IntelliJ Plugin developed for  the Univesity Of Birmingham Dubai (UOBD) Google Developers Group on Campus (GDGoC) Bounty Challenge.

💡 The Concept: Gamifying the Debugging Loop
Beginner developers often find IDE error messages cryptic and discouraging. Our team conceptualized and prototyped an IntelliJ plugin that transforms standard error logs into an interactive learning experience:

AI-Generated MCQs: Leverages the OpenAI API to turn real-time stack traces into multiple-choice reasoning prompts.
Active Learning: Instead of just "fixing" the code, users are challenged to identify the root cause to earn points.
Leaderboard Integration: Introduced a competitive concept to promote consistent coding habits and engagement.
